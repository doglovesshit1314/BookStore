## BookStore
A JavaWeb project based on SSH framework who can manage books,search book infomation,buy books.

## Introduction

**Base on the MVP SSH framework .**

* foreground
	* user login/register
	* search book
	* place order
	* shopping cart
	* manage order
	* personal management
* background
	* manager login
	* check orders
	* delivery
	* add/delete books
	* revise infomation
	* personal management
